Detailed Description

Consumer interface module for the Canteen Management System.

This file provides a standalone consumer-side interface where users can:

  • View menu items
  • Place orders
  • Cancel last order
  • View past orders

This module works with shared data structures such as Menu, Consumer, OrderQueue, and OrderStack.
